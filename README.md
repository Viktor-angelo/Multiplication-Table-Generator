#  Multiplication Table Generator

A simple and responsive web application that generates multiplication tables based on user input.

##  Features

- Generate multiplication tables dynamically
- Input validation to prevent crashes
- Limit control (max 100 iterations)
- Light/Dark mode toggle
- Responsive design (mobile-friendly)

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

##  Preview

![App Screenshot](./screenshot.png)

##  How It Works

1. Enter a number
2. Choose how far the multiplication should go
3. Click "Calculate"
4. View the generated multiplication table instantly

##  Validation

- Maximum limit: **100**
- Prevents invalid or negative inputs
- Avoids browser performance issues
